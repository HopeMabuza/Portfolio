SecureNFT

NFT-gated dApp deployed on Sepolia. An ERC1155 contract issues soulbound Rose NFTs that act as access keys, only Rose holders can enter the garden and mint Lily NFTs. Access is enforced server-side via an Express backend that queries the blockchain directly, preventing frontend bypass. Features a React frontend with canvas particle animations and an SVG garden scene.
